# خطة العمل — تطوير شامل لبوابة التدريب

طلبك ضخم ومتشعّب جداً (7 محاور كبرى). قبل ما أبدأ، عاوز أتأكد من الترتيب والأولويات علشان أسلّمك شغل متين بدل ما أكسر حاجة شغالة.

---

## 1. إصلاح حرج: ظهور طلبات الانضمام عند الأدمن
- فحص `enrollments` query في `admin.tsx` + RLS + التريجر
- التأكد إن `user_id` و `course_id` بيتسجلوا صح، وإن الإشعار بيوصل
- اختبار End-to-End: متدرب يطلب → يظهر فوراً عند الأدمن

## 2. التسجيل والصلاحيات
- Pop-up "تم إرسال رابط التفعيل" بعد signup
- وضع Teaser: المتدرب غير المعتمد يشوف عناوين المحاضرات فقط مع 🔒
- زرار "انسحاب" يظهر فقط لو الطلب pending
- في الأدمن: "حذف متدرب من الكورس" + "قفل/حظر مؤقت" (عمود `blocked` جديد على `enrollments`)

## 3. هيكلة الكورس
- إعادة تسمية "الباب" → "المحاضرة" في كل واجهات الأدمن
- الصور/الروابط تظهر كأزرار "تصفح المحتوى" بدل URL خام
- **تصنيفات شجرية**: جدول `course_categories` جديد، والمحاضرات تترتب تحته
- جدول مواعيد ذكي: لما تضيف session، Droplist بيسحب المحاضرات الموجودة لربطها
- دمج `online_url` (الكورس) و رابط المنصة في حقل واحد "رابط الكورس"
- حقل `total_hours` جديد على `courses`

## 4. رفع الاختبارات
- جدول `assignments` + `assignment_submissions` جديد
- المتدرب يرفع من الداشبورد + من داخل صفحة الكورس
- Drive link من الأدمن + خانة رفع للمتدرب جنبه

## 5. داشبورد المتدرب (كروت)
- عدد الكورسات (نشط/مكتمل)
- إجمالي الساعات (مجموع `total_hours` للكورسات المعتمد فيها)
- الموقف المالي (أقساط مستحقة + اسم الكورس)

## 6. الشهادات + LinkedIn
- حقل `certificate_url` موجود بالفعل — تحسين الـ UX
- إيميل + إشعار حماسي عند الرفع (trigger موجود — يتوسّع)
- زرار "مشاركة على LinkedIn" مع نص ديناميكي عربي/إنجليزي جاهز للنسخ

## 7. الترجمة الاحترافية
- نظام i18n موجود بالفعل (`src/lib/i18n.tsx`) — توسعته ليشمل كل واجهات الأدمن/المتدرب
- **الترجمة الديناميكية للمحتوى**: server function تستخدم Lovable AI Gateway (Gemini) لترجمة وصف الكورس/المحاضرة على الطاير لو المستخدم اختار EN

---

## تقدير الحجم والمخاطر
الطلب ده فعلياً = **3-4 أيام شغل متواصل** لو اتعمل صح، وفيه:
- ~6 migrations جديدة (تصنيفات، assignments، blocked، total_hours، إعادة هيكلة)
- ~12 تعديل كبير في `admin.tsx` و `portal.tsx`
- 4-5 مكونات React جديدة (داشبورد كروت، LinkedIn share, locked module, assignments)
- 1 server function للترجمة الذكية

**اقتراحي:** أقسّم الشغل على 3 جولات علشان كل جولة تتظبط وتتجرّب قبل ما ننتقل للي بعدها، بدل ما أعمل كل حاجة دفعة واحدة وتطلع أخطاء صعب نلاقيها.
